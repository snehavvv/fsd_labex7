API Documentation

Available Endpoints

GET /api/menu

Get all menu items (supports query parameters)

GET /api/categories

Get all menu categories

GET /api/cuisines

Get all cuisine types

Query Parameters

name Filter by menu item name
category Filter by category
cuisine Filter by cuisine type
minPrice Minimum price filter
maxPrice Maximum price filter
available Filter by availability (true/false)

Example Usage

/api/menu?category=Main Course&cuisine=Indian&minPrice=10&maxPrice=25&available=true
