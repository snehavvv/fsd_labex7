API Documentation

Available Endpoints

GET /api

Get all products (supports query parameters)

GET /api/products

Get all products

GET /api/categories

Get all categories

GET /api/brands

Get all brands

GET /api/products/:id

Get product by ID

Query Parameters

name Filter by product name
category Filter by category
brand Filter by brand
minPrice Minimum price filter
maxPrice Maximum price filter
inStock Filter by stock status (true/false)

Example Usage

/api?category=Electronics&minPrice=100&maxPrice=500&inStock=true
